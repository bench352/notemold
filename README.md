# NoteMold

**NoteMold** is a **Markdown-compatible notation system** that turns a regular Markdown/plaintext document into a dynamic template. It adopts a syntax that is easy to read and write in natural language, making it digestible for both humans and LLMs.

## Syntax guide

You can learn how to create a template using NoteMold syntax using this [Syntax guide](./SYNTAX_GUIDE.md).

## Agent Skills

[Agent Skills](https://agentskills.io/home) are created to provide compatible AI Agents the capability to correctly fill the templates and complete the document for you as you intended. Agent Skills are available in the subdirectories of this repository. The available Agent Skills include:

- **NoteMold Formulator** (`notemold-formulator/`): Skills to turn a user-provided Markdown/plaintext template into a finalized, completed document following the NoteMold syntax defined in the template.
